# Online-Book-Store
A full-stack web application that enables multiple sellers to list and sell books and allows users to browse, review, and purchase books seamlessly.

-> Implemented a secure authentication system with session handling and MFA to prevent unauthorized access.

-> Designed a review and rating module enabling buyers to make informed decisions and help sellers improve service quality.

-> Implemented seller-independent order processing, allowing each seller to manage orders separately.

-> Designed a normalized MySQL database schema to reduce redundancy and improve query performance.
